# Challenge #2: Write all the test names you would write 

Write a comprehensive list of all the test names you would write for the class "GenerateCsv". When done, make a pull request with the title "challenge 2".

## Details

The code is located in the "code" folder. Write down your solution in the file solution.txt.

### Scoring

You get points for: 

* clear test names
* the number of behaviors covered.

You will NOT get points if you write the tests or for tests checking only the private methods.

You will get:

* 0 points if no pull request 
* 1 point if you make a pull request even if there's no solution
* 2-20 points if you make a pull request with a partial or complete solution
